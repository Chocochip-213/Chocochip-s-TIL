# Chocochip's TIL

공부한 내용을 꾸준히 정리하고, 문제 풀이와 개념 학습 기록을 모아두는 TIL 저장소입니다.
알고리즘 풀이, 자료구조/CS 정리, 그리고 학습 과정에서 남긴 메모들을 함께 관리합니다.

## 기록하는 내용

- 알고리즘 문제 풀이
- 자료구조 / CS 개념 정리
- Git, Python, Java 학습 내용
- B형 대비 문제 풀이 기록
- 프로젝트를 진행하며 정리한 메모

## 폴더 구조

```text
TIL/
  git/                  # Git 학습 정리
  data-structure/       # 자료구조 개념 정리
  algorithm-notes/      # 알고리즘 학습 노트
  project-notes/        # 프로젝트 관련 메모

Algorithm/
  SWEA/                 # SWEA 문제 풀이
  SSAFY강의/             # SSAFY 강의 실습 풀이
  백준/                  # 백준 문제 풀이
  Java_Practice/        # Java 알고리즘 연습
  CPP_Practice/         # C++ 알고리즘 연습
  B형 대비 기출/          # SW 역량테스트 B형 기출 풀이
```

## 정리 방식

- 개념 정리는 주제별로 분류합니다.
- 알고리즘 문제는 플랫폼/주제 기준으로 분류합니다.
- 같은 문제를 다시 풀었다면 개선 과정이 보이도록 기록합니다.
- 불필요한 파일보다는 다시 찾아보기 쉬운 구조를 우선합니다.

## 현재 집중하는 주제

- 알고리즘 문제 풀이
- 자료구조와 그래프
- 구현/시뮬레이션 문제 정리
- B형 대비 문제 복습
